Movie Recommender App

After downloading run front.py


embed.py is used for embedding the cleaned data set from https://www.kaggle.com/datasets/asaniczka/tmdb-movies-dataset-2023-930k-movies (the algorithm cleans unused rows and finds the movies that match the prompt). Then the data is saved in the files from the drive below. backend.py uses this data that front.py will show

For the movie posters i have used the TMDB database
